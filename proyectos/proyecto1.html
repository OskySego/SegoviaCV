Proyecto 1: Mi Primer Portafolio

La base de mi viaje en el desarrollo web Front-End.

Detalles del Proyecto

Este proyecto representa mi primera incursión en el desarrollo web. Aquí apliqué los fundamentos de HTML para estructurar el contenido y CSS para darle un estilo básico. Fue una etapa crucial para entender cómo se construyen las páginas web desde cero.

Aprendí sobre la estructura semántica de HTML, cómo usar etiquetas para encabezados, párrafos, listas y tablas. En CSS, experimenté con selectores, propiedades de color, tipografía y el modelo de caja. Aunque sencillo, este proyecto sentó las bases para mis futuros aprendizajes y mejoras.

Tecnologías Utilizadas:

  • HTML5: Para la estructura y el contenido.
  • CSS3: Para el diseño y la presentación.
